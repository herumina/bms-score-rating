発狂難易度表のAAA、MAX-埋めのモチベを上げるために作りました。</br>

LR2の場合は、名前.db　orajaの場合はscorelog.dbを読み込ませるとレーティングが出てきます。</br>

譜面定数については、hakkyoudata.csvの「difficulty」に書いてあります。</br>
基本的にLR2IRの上位スコア(25,50,75,100位の平均値)を基に決め、「上から殴るとスコアが取りやすい譜面(特に腕押し)」や、「AAAを取るまでは簡単な譜面」などについては体感で調整しました。</br>
皆さんのレーティング計測結果を参考に調整していきます。</br>

現状のレーティング計算式はこんな感じです。</br>
今後変動する可能性があります。</br>

基礎値</br>
(譜面定数)×(スコアレート(MAXで1.00))</br>

補正値</br>
AAAボーダー(8÷9×100)との差分×0.5で増減</br>
AAAボーナス +1.5</br>
MAX-ボーナス +1.5(AAAと合わせて+3.0)</br>

例</br>
★19	点、線、面、立体 多胞体 譜面定数 14</br>
スコア 96.32%</br>

基礎値</br>
14*0.9632=13.4848</br>

補正値</br>
AAAボーダーとの差分×0.5 3.7156</br>
AAAボーナス +1.5</br>
MAX-ボーナス +1.5</br>

レート値 20.2004</br>

全体レートにボーナス</br>
AAAボーナス AAAの数だけ+0.1</br>
MAX-ボーナス MAX-の数だけ+0.1</br>


注意点</br>
・バイブコーディングで作ったのでコードの中身はそんなによく分かってません。</br>
