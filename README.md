発狂難易度表のAAA、MAX-埋めのモチベを上げるために作りました、

LR2の場合は、名前.db　orajaの場合はscorelog.dbを読み込ませるとレーティングが出てきます。

現状のレーティング計算式はこんな感じです。
今後変動する可能性があります。(97%、98%ボーナスを追加するか検討中)

譜面定数については、hakkyoudata.csvに書いてあります。
基本的にLR2IRの上位スコア(25,50,75,100位の平均値)を基に決め、「上から殴るとスコアが取りやすい譜面(特に腕押し)」や、「MAX-から詰めるのが難しい譜面」などについては体感で調整しました。
おそらく何かしら明らかな稼ぎ譜面や詐称譜面があると思いますがご了承ください。
皆さんのレーティング計測結果を参考に調整していきます。

基礎値
(譜面定数)×(スコアレート(MAXで1.00))

補正値
AAAボーダー(88.8888)との差分×0.5で増減
AAAボーナス +1.5
MAX-ボーナス +1.5(AAAと合わせて+3.0)
99％ボーナス +2.0(AAA,MAX-と合わせて+5.0)

例
★19	点、線、面、立体 多胞体 譜面定数 14
スコア 96.32%

基礎値
14*0.9632=13.4848

補正値
AAAボーダーとの差分×0.5 3.7156
AAAボーナス +1.5
MAX-ボーナス +1.5

レート値 20.2004
